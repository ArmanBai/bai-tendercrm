# BAi TenderCRM

**BAi TenderCRM** is a tender CRM and analytics product concept developed by BAi Core for procurement workflow automation, supplier decision support and tender-related business analytics.

## Current status

The product is currently in ownership transfer and documentation preparation stage.

Source code is not published yet.

## Product idea

TenderCRM is designed to help companies manage tender participation more systematically:

- track tender opportunities;
- analyze customers and suppliers;
- evaluate risks before participation;
- manage tender workflow;
- prepare documentation;
- support decision-making with analytics and AI-assisted tools.

## Planned modules

- Supplier workspace
- Customer workspace
- Investor analytics
- Tender analytics dashboard
- AI assistant for tender analysis
- Risk scoring
- Document assistant
- CRM pipeline for tender participation

## Target market

Kazakhstan and CIS markets, with focus on B2B and B2G procurement processes.
